Deploy Monitoring Agent

Securely onboard a new Windows server or workstation using the professional agent installer.

1. Generate Onboarding Token

Before launching the installer, generate a secure one-time registration token. The token authorizes the installer to configure the daemon credentials.

2. Download & Run Installer

Copy the onboarding token, download the installer package, and execute the installation. Paste the token when prompted by the wizard.

15m expiry
⏳ Token expires in 15:00. Complete the installation before this time.
