Deploy Agent

Install the monitoring agent on any target node in seconds.

How It Works

1

Copy the PowerShell command below

2

Run in PowerShell as Administrator on the target node

3

The agent installs, connects, and appears in your Devices

PowerShell Install Command

⚠️ Run as Administrator. The script downloads the agent binary, writes workspace configuration details, and registers a background startup daemon.

Download Script Directly

If you prefer manual execution, download the raw installer command file and run it inside your environment.

Uninstall Agent

Run this command in PowerShell as Administrator to completely remove the agent background scheduled task and installation files:

Unregister-ScheduledTask -TaskName ProactiveITAgent -Confirm:$false; Remove-Item -Recurse -Force "$env:ProgramData\ProactiveITAgent"
